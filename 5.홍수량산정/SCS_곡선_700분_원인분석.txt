■ 현상
  배치 분석 후 GUI 그래프에서 Clark·Nakayasu 수문곡선은 400~500분 부근에서
  종료되는 반면, SCS 수문곡선만 700분 근방까지 연장되어 표시됨.

────────────────────────────────────────────────────────────────────────────────
■ 원인
────────────────────────────────────────────────────────────────────────────────

1. SCS UH의 무차원 테이블 구조
   ─────────────────────────────
   SCS 합성단위도(get_scs_uh)는 t/Tp = 0 ~ 5.0 범위의 무차원 테이블로 정의됨.
   테이블 후반부(감수부):

     t/Tp  │  q/Qp
     ──────┼────────
     4.0   │  0.011
     4.5   │  0.005
     5.0   │  0.000   ← 여기서 정확히 0

   SCS UH의 실제 시간 길이 = 5 × Tp

2. Tp 계산 예시
   ─────────────
   SCS_TC = 2 hr,  crit_tr = 180분 (3 hr) 일 때:
     Tp = 0.6 × SCS_TC + tr/2
        = 0.6 × 2    + 3/2
        = 1.2 + 1.5  = 2.7 hr

   SCS UH 길이 = 5 × 2.7 hr = 13.5 hr = 810분

3. 합성곱(Convolution) 결과 길이
   ────────────────────────────────
   합성곱 결과 총 길이 = 강우 지속기간 + UH 길이 - 1
                      = 180분 + 810분 - 1 = 989분

   trailing zero 제거(threshold = 0.001 cms) 적용 후 → 약 700분 근방에서 끊김

4. Clark·Nakayasu가 더 짧게 표시되는 이유
   ──────────────────────────────────────────
   Clark와 Nakayasu는 지수감소형(exponential recession) 감수부를 가짐.

   Clark  : 선형저류 방정식 → O(t) ∝ exp(-dt/R) 형태로 급감
   Nakayasu: 지수형 분기 감수 공식 (0.3^ratio) 형태

   → 지수 감쇠가 빠르게 진행되어 0.001 cms threshold 아래로 일찍 도달
   → trailing zero 제거가 400~500분 구간에서 이미 발생

   SCS는 t/Tp = 4.0 ~ 5.0 구간에서 선형보간으로 천천히 0에 접근하므로
   같은 threshold 기준으로 훨씬 늦게 제거됨.

────────────────────────────────────────────────────────────────────────────────
■ 물리적 타당성
────────────────────────────────────────────────────────────────────────────────
   수문학적으로는 정확한 결과임.
   SCS UH 자체가 Clark·Nakayasu보다 더 긴 감수부를 가지는 방법이므로
   수문곡선이 더 오래 지속되는 것은 이론에 부합함.

   다만 그래프 표현 상 시각적 불균형이 발생하는 문제가 있음.
