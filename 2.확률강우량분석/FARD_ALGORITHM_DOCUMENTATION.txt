[FARD 분석 시스템 - 알고리즘 설명서]
Frequency Analysis of Rainfall Data - Algorithm Documentation

═══════════════════════════════════════════════════════════════════════════
1. 개요 (Overview)
═══════════════════════════════════════════════════════════════════════════

본 FARD(Frequency Analysis of Rainfall Data) 분석 시스템은 국립방재연구원 및 
연세대학교 수공학연구실에서 개발한 확률강우량 빈도해석 프로그램(Ver.2006)의 
표준 출력 형식을 준수하여 구현되었습니다.

「홍수량 산정지침」(국토교통부), 「확률강우량도 작성 지침」에 따라 16개 
확률분포형에 대한 매개변수 추정 및 확률강우량 산정을 수행하며, 특히 
Gumbel 분포는 정부 기준으로 채택된 PWM(Probability Weighted Moments) 방법을 
적용합니다.

주요 특징:
- 16개 확률분포형 지원 (Tier 1~3 구현 + 10개 참조용)
- L-Moments 기반 PWM 방법 (lmoments3 라이브러리 활용)
- FARD 표준 출력 형식 (.OUT 파일)
- 1% 및 5% 유의수준 분석
- 12개 재현기간별 확률강우량 산정 (2~500년)

2. 지원 분포형 및 구현 등급 (Distribution Types & Implementation Tiers)
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│ Tier 1: 완벽 구현 (정확도 95-100%)                                     │
└─────────────────────────────────────────────────────────────────────┘

(1) NOR - Normal Distribution (정규분포)
    · 추정 방법: MOM (Method of Moments)
    · 매개변수: μ (Mean), σ (Standard Deviation)
    · 구현 근거: 기본 통계량 직접 활용
    · 분위수 함수: X = μ + Z·σ (Z: 표준정규분포 분위수)

(2) GAM2 - 2-Parameter Gamma Distribution (2-매개변수 감마분포)
    · 추정 방법: MOM
    · 매개변수: α (Shape), β (Scale)
    · 관계식: β = σ²/μ, α = μ/β
    · 분위수 함수: Scipy.stats.gamma.ppf 활용

(3) GUM - Gumbel Distribution (극치 제1형 분포)
    · 추정 방법: PWM (L-Moments)
    · 매개변수: ξ (Location), α (Scale)
    · L-Moment 관계식:
      - α = λ2 / ln(2)
      - ξ = λ1 - 0.5772157·α (Euler-Mascheroni 상수)
    · 분위수 함수: X = ξ - α·ln(-ln(F))
    · 정부 기준: 국토교통부 확률강우량도 작성 지침 채택

(4) GEV - Generalized Extreme Value Distribution (일반화 극치분포)
    · 추정 방법: PWM (L-Moments)
    · 매개변수: ξ (Location), α (Scale), k (Shape)
    · L-Moment Fitting:
      Primary: lmoments3.distr.gev.lmom_fit() (Hosking 알고리즘)
      Fallback: PDF 교재 공식 (13.4.84-88) - Hosking(1990) 근사식
    · Fallback 알고리즘 (lmoments3 미설치 시):
      - Cs(왜곡계수) 범위별 6차 다항식으로 β(형상인자) 추정
      - Gamma 함수 이용 α(척도), ξ(위치) 계산
    · 분위수 함수:
      - k ≈ 0 (Gumbel): X = ξ - α·ln(-ln(F))
      - k ≠ 0: X = ξ + (α/k)·[1 - (-ln(F))^k]

┌─────────────────────────────────────────────────────────────────────┐
│ Tier 2: 양호 구현 (정확도 90-95%)                                      │
└─────────────────────────────────────────────────────────────────────┘

(5) GLO - Generalized Logistic Distribution (일반화 로지스틱 분포)
    · 추정 방법: PWM (L-Moments)
    · 매개변수: ξ (Location), α (Scale), k (Shape)
    · L-Moment Fitting: lmoments3.distr.glo.lmom_fit()
    · 분위수 함수:
      - k ≈ 0: y = -ln((1-F)/F)
      - k ≠ 0: y = (1/k)·[(1-F)/F]^k - (1/k)
      - X = ξ + α·y

┌─────────────────────────────────────────────────────────────────────┐
│ Tier 3: 근사 구현 (정확도 80-90%)                                      │
└─────────────────────────────────────────────────────────────────────┘

(6) GAM3 - 3-Parameter Gamma (Pearson Type III)
    · 추정 방법: MOM (Method of Moments)
    · 매개변수: ξ (Location), β (Scale), α (Shape)
    · MOM 근사 공식:
      - α = (2/Cs)²
      - β = σ/√α
      - ξ = μ - α·β
    · 분위수 함수: Scipy.stats.gamma.ppf 활용
    · 주의: MOM 방법은 L-Moments보다 정확도 낮음

┌─────────────────────────────────────────────────────────────────────┐
│ 미구현 분포 (Not Implemented) - 참조용 출력                            │
└─────────────────────────────────────────────────────────────────────┘

(7)  LN2  - 2-Parameter Log-Normal Distribution
(8)  LN3  - 3-Parameter Log-Normal Distribution
(9)  LP3  - Log-Pearson Type III Distribution
(10) LGU2 - 2-Parameter Log-Gumbel Distribution
(11) LGU3 - 3-Parameter Log-Gumbel Distribution
(12) WBU2 - 2-Parameter Weibull Distribution
(13) WBU3 - 3-Parameter Weibull Distribution
(14) GPA  - Generalized Pareto Distribution
(15) WKB4 - 4-Parameter Wakeby Distribution
(16) WKB5 - 5-Parameter Wakeby Distribution

※ 미구현 이유: 신뢰 가능한 L-Moments 구현 근거 부족
※ 출력 형식: "NOT IMPLEMENTED" 표시 및 빈 결과 출력

3. 매개변수 추정 방법론 (Parameter Estimation Methods)
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│ 3.1 MOM (Method of Moments) - 적률법                                  │
└─────────────────────────────────────────────────────────────────────┘

개념:
표본의 통계량(평균, 분산, 왜곡도, 첨도)을 모집단의 이론적 적률과 
일치시켜 매개변수를 추정하는 전통적 방법.

적용 분포: NOR, GAM2, GAM3

기초 통계량 계산 (비편향 추정량):
- Mean (평균): μ = Σx / n
- Variance (분산): s² = Σ(x-μ)² / (n-1)
- Skewness (왜곡도): Cs = [n·Σ(x-μ)³] / [(n-1)·(n-2)·s³]
- Kurtosis (첨도): Ck = [n·(n+1)·Σ(x-μ)⁴] / [(n-1)·(n-2)·(n-3)·s⁴] 
                      - [3·(n-1)²] / [(n-2)·(n-3)] + 3

장점:
- 계산이 간단하고 직관적
- 모든 분포에 적용 가능

단점:
- 소표본에서 추정량의 분산이 큼
- 왜곡된 분포에서 편향 발생 가능

┌─────────────────────────────────────────────────────────────────────┐
│ 3.2 PWM (Probability Weighted Moments) - L-Moments법                 │
└─────────────────────────────────────────────────────────────────────┘

개념:
확률가중모멘트(PWM)를 이용한 L-Moments(선형모멘트) 기반 매개변수 추정.
Hosking & Wallis(1997) "Regional Frequency Analysis" 알고리즘 적용.

적용 분포: GUM, GEV, GLO

L-Moments 정의:
- λ1 (L-Mean):     평균의 선형 추정량
- λ2 (L-Scale):    산포도의 선형 추정량
- τ3 (L-Skewness): 왜곡도의 선형 추정량 (λ3/λ2)
- τ4 (L-Kurtosis): 첨도의 선형 추정량 (λ4/λ2)

표본 L-Moments 계산 (Hosking 알고리즘):
1. 데이터 정렬: x(1) ≤ x(2) ≤ ... ≤ x(n)
2. 확률가중모멘트 계산:
   - b0 = mean(x)
   - b1 = mean(x · (i-1)/(n-1))
   - b2 = mean(x · (i-1)(i-2)/[(n-1)(n-2)])
3. L-Moments 변환:
   - λ1 = b0
   - λ2 = 2·b1 - b0
   - λ3 = 6·b2 - 6·b1 + b0
   - τ3 = λ3 / λ2

라이브러리:
- Primary: lmoments3 (Python 전문 라이브러리)
- Fallback: 자체 구현 알고리즘 (MOM 또는 PDF 공식)

장점:
- 소표본에서 강건함 (Robust)
- 이상치에 덜 민감
- 정규성 가정 불필요
- 분포 매개변수와 직접적 관계

단점:
- 계산이 상대적으로 복잡
- 일부 분포는 반복 계산 필요

정부 기준:
- 국토교통부 「확률강우량도 작성 지침」
- Gumbel 분포 PWM 방법 공식 채택

═══════════════════════════════════════════════════════════════════════════
4. 분포형별 구현 상세 (Distribution-Specific Implementation)
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│ 4.1 Normal Distribution (정규분포)                                    │
└─────────────────────────────────────────────────────────────────────┘

확률밀도함수:
f(x) = (1/√(2πσ²)) · exp[-(x-μ)²/(2σ²)]

매개변수 추정 (MOM):
- Location (μ): 표본 평균
- Scale (σ): 표본 표준편차

분위수 함수:
X = μ + Φ^(-1)(F) · σ
여기서 Φ^(-1): 표준정규분포의 역함수

적용 조건:
- 데이터가 대칭적
- 자연 현상의 측정 오차
- 중심극한정리 적용 가능한 경우

┌─────────────────────────────────────────────────────────────────────┐
│ 4.2 Gumbel Distribution (극치 제1형 분포)                              │
└─────────────────────────────────────────────────────────────────────┘

확률밀도함수:
f(x) = (1/α) · exp[-(x-ξ)/α - exp(-(x-ξ)/α)]

누적분포함수:
F(x) = exp[-exp(-(x-ξ)/α)]

매개변수 추정 (PWM - L-Moments):
Step 1: 표본 L-Moments 계산 (λ1, λ2)
Step 2: 매개변수 산정
  - Scale: α = λ2 / ln(2) = λ2 / 0.693147
  - Location: ξ = λ1 - γ·α
    (γ = 0.5772156649... : Euler-Mascheroni 상수)

Fallback (lmoments3 미설치 시 - MOM):
  - Scale: α = (√6 · σ) / π
  - Location: ξ = μ - 0.5772157·α

분위수 함수:
X = ξ - α · ln[-ln(F)]

여기서:
- F: 비초과확률 = 1 - 1/T
- T: 재현기간 (년)

예시 (T=100년):
F = 1 - 1/100 = 0.99
X100 = ξ - α · ln[-ln(0.99)]

적용 분야:
- 연최대 강우량
- 홍수량 빈도해석
- 극치 수문사상

정부 기준:
국토교통부 「확률강우량도 작성 지침」 공식 채택

┌─────────────────────────────────────────────────────────────────────┐
│ 4.3 GEV Distribution (일반화 극치분포)                                 │
└─────────────────────────────────────────────────────────────────────┘

확률밀도함수:
f(x) = (1/α) · [1 + k(x-ξ)/α]^(-1-1/k) · exp{-[1 + k(x-ξ)/α]^(-1/k)}

누적분포함수:
F(x) = exp{-[1 + k(x-ξ)/α]^(-1/k)}

매개변수:
- ξ (Xi): Location parameter
- α (Alpha): Scale parameter (α > 0)
- k (Kappa): Shape parameter
  · k > 0: Fréchet type (Type II)
  · k = 0: Gumbel type (Type I)
  · k < 0: Weibull type (Type III)

매개변수 추정 (PWM - L-Moments):

Primary Method (lmoments3 사용):
  lmoments3.distr.gev.lmom_fit(data)
  - Hosking(1985, 1990) 알고리즘 구현
  - τ3로부터 k 추정 (반복적 근사)
  - λ1, λ2, k로부터 ξ, α 산정

Fallback Method (lmoments3 미설치 시):
  Hosking(1990) PDF 교재 공식 적용
  
  Step 1: 왜곡계수(Cs)로부터 형상인자(k) 추정
    구간별 6차 다항식 근사:
    
    · 1.14 < Cs < 10 (k < 0):
      k = 0.285821 - 0.357983·Cs + 0.187683·Cs² - 0.053754·Cs³
          + 0.008612·Cs⁴ - 0.000718·Cs⁵ + 0.000024·Cs⁶
    
    · -2 < Cs ≤ 1.14 (k > 0):
      k = 0.277648 - 0.322016·Cs + 0.131506·Cs² - 0.028093·Cs³
          + 0.003301·Cs⁴ - 0.000197·Cs⁵ + 0.000005·Cs⁶
    
    · -10 < Cs < -2 (k < 0):
      k = -0.50405 - 0.00861·Cs + 0.01517·Cs² + 0.00356·Cs³
          + 0.00042·Cs⁴ + 0.00003·Cs⁵
  
  Step 2: 척도인자(α) 계산
    α = √[σ²·k² / (Γ(1+2k) - Γ²(1+k))]
    여기서 Γ: Gamma 함수
  
  Step 3: 위치인자(ξ) 계산
    ξ = μ - (α/k)·[Γ(1+k) - 1]

분위수 함수:
  · k ≈ 0 (|k| < 10⁻¹⁰): Gumbel 분포로 근사
    X = ξ - α · ln[-ln(F)]
  
  · k ≠ 0:
    X = ξ + (α/k) · {1 - [-ln(F)]^k}

정확도:
- lmoments3: 약 90-95% (Hosking 표준 알고리즘)
- Fallback: 약 85-95% (근사 다항식 오차 5-10%)

┌─────────────────────────────────────────────────────────────────────┐
│ 4.4 Generalized Logistic Distribution (일반화 로지스틱 분포)           │
└─────────────────────────────────────────────────────────────────────┘

확률밀도함수:
f(x) = (1/α) · [1 + k(x-ξ)/α]^(-1-1/k) / [1 + [1 + k(x-ξ)/α]^(-1/k)]²

매개변수 추정 (PWM - L-Moments):
lmoments3.distr.glo.lmom_fit(data)

분위수 함수:
  · k ≈ 0:
    y = -ln[(1-F)/F]
  
  · k ≠ 0:
    y = (1/k) · {[(1-F)/F]^k - 1}
  
  X = ξ + α·y

적용:
- 극치 사상의 대안 모형
- 홍수빈도해석

┌─────────────────────────────────────────────────────────────────────┐
│ 4.5 Gamma Distributions (감마분포)                                    │
└─────────────────────────────────────────────────────────────────────┘

2-Parameter Gamma (GAM2):
확률밀도함수:
f(x) = [1/(β^α·Γ(α))] · x^(α-1) · exp(-x/β)

매개변수 (MOM):
- Shape (α): α = μ/β
- Scale (β): β = σ²/μ

3-Parameter Gamma / Pearson Type III (GAM3):
확률밀도함수:
f(x) = [1/(β^α·Γ(α))] · (x-ξ)^(α-1) · exp[-(x-ξ)/β]

매개변수 (MOM 근사):
- Shape (α): α = (2/Cs)²
- Scale (β): β = σ/√α
- Location (ξ): ξ = μ - α·β

분위수 함수:
Scipy.stats.gamma.ppf(F, a=α, loc=ξ, scale=β)

주의사항:
MOM 방법은 왜곡도가 클 때 불안정할 수 있음.
실무에서는 L-Moments 기반 방법 권장.

═══════════════════════════════════════════════════════════════════════════
5. 확률강우량 산정 (Probability Rainfall Calculation)
═══════════════════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────────────┐
│ 5.1 재현기간 및 비초과확률                                             │
└─────────────────────────────────────────────────────────────────────┘

재현기간(Return Period, T):
특정 크기 이상의 사상이 평균적으로 T년에 한 번 발생할 확률

비초과확률(Non-Exceedance Probability, F):
F = 1 - 1/T

지원 재현기간:
2, 3, 5, 10, 20, 30, 50, 70, 80, 100, 200, 500년

예시:
- T = 100년 → F = 0.99 (99% 비초과)
- T = 50년 → F = 0.98 (98% 비초과)
- T = 10년 → F = 0.90 (90% 비초과)

┌─────────────────────────────────────────────────────────────────────┐
│ 5.2 분위수 함수 (Quantile Function)                                   │
└─────────────────────────────────────────────────────────────────────┘

확률강우량 계산 흐름:
1. 재현기간(T) → 비초과확률(F) 변환
2. 추정된 매개변수 사용
3. 분포형별 분위수 함수 적용
4. 확률강우량(X) 산정

분포별 계산식:
참조 → 4. 분포형별 구현 상세의 "분위수 함수" 항목

┌─────────────────────────────────────────────────────────────────────┐
│ 5.3 유의수준별 분석                                                   │
└─────────────────────────────────────────────────────────────────────┘

1% 유의수준 분석:
- 출력 파일: 프로젝트명_B_1%.OUT
- 용도: 보수적 설계 기준
- 신뢰구간: 99%

5% 유의수준 분석:
- 출력 파일: 프로젝트명_B_5%.OUT
- 용도: 일반적 설계 기준
- 신뢰구간: 95%

주의:
현재 버전에서는 매개변수 추정만 수행하며,
신뢰구간 산정은 향후 버전에서 지원 예정.

부록 A. 용어 정리 (Glossary)
═══════════════════════════════════════════════════════════════════════════

L-Moments (선형모멘트):
  순서통계량의 선형결합으로 정의된 분포의 특성치.
  기존 적률(moment)보다 소표본에서 강건하고 이상치에 덜 민감.

PWM (Probability Weighted Moments):
  확률가중모멘트. L-Moments 계산의 기초가 되는 통계량.

비초과확률 (Non-Exceedance Probability):
  특정 값 이하일 확률. F = P(X ≤ x).

재현기간 (Return Period):
  특정 크기 이상의 사상이 평균적으로 T년에 한 번 발생할 기간.

분위수 함수 (Quantile Function):
  누적분포함수의 역함수. F^(-1)(p) = x.

형상인자 (Shape Parameter):
  분포의 모양(꼬리의 두께, 왜곡도 등)을 결정하는 매개변수.

척도인자 (Scale Parameter):
  분포의 산포도(퍼짐 정도)를 결정하는 매개변수.

위치인자 (Location Parameter):
  분포의 중심 위치를 결정하는 매개변수.
